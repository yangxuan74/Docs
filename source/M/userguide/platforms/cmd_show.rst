.. _cmd_platform_show:

pio platform show
=================

.. contents::

Usage
-----

.. code-block:: bash

    pio platform show PLATFORM


Description
-----------

Show details about :ref:`platforms`


Examples
--------

.. code::

    > pio platform show atmelavr
