pio update
==========

.. contents::

Usage
-----

.. code-block:: bash

    pio update [OPTIONS]


Description
-----------


Options
-------

.. program:: pio update

.. option::
    --core-packages

Update only the core packages

.. option::
    -c, --only-check

DEPRECATED. Please use ``--dry-run`` instead.


.. option::
    --dry-run

Do not update, only check for the new versions
