.. _cmd_upgrade:

pio upgrade
===========

.. contents::

Usage
-----

.. code-block:: bash

    pio upgrade

Description
-----------

Check or upgrade PlatformIO to the latest version

Options
-------

.. program:: pio upgrade

.. option::
    --dev

Use development branch.

Examples
--------

.. code::

    > pio upgrade

    You are up-to-date!
    PlatformIO x.x.x is currently the newest version available.

    # If you have problem with permissions try:
    > sudo pio upgrade
