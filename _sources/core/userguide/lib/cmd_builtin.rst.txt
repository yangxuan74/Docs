.. _cmd_lib_builtin:

pio lib builtin
===============

.. contents::

Usage
-----

.. code-block:: bash

    pio lib builtin [OPTIONS]


Description
-----------

Options
-------

.. program:: pio lib builtin

.. option::
    --storage

List libraries from specified storages. For example, ``framework-arduinoavr``.

.. option::
    --json-output

Return the output in `JSON <http://en.wikipedia.org/wiki/JSON>`_ format
